# skill_fetch_trends

## Purpose
Fetch trending topics/hashtags for a given niche/platform (minimal stub for now).

## Inputs
- platform (e.g. youtube, tiktok, twitter)
- niche (e.g. tech, AI, health)

## Output
A list of trend objects, e.g.
- title
- reason/angle
- suggested hashtags
